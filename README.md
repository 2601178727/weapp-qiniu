# 微信小程序－七牛图片上传

### 说明：

实现了直接从客户端上传图片到七牛的功能，包含生成uptoken的服务端。

### 数据接口:

- https://aime.getweapp.com/qiniu/uptoken
- 自己uptoken服务端api

### 目录结构：

- client — 客户端
- server — 获取uptoken服务端

### 开发环境：

微信web开发者工具 v0.11.122100
